# Swifi

**Swifi** is a lightweight CLI tool to check wifi speeds.

Faster, easier and cooler than web alternatives.

## Quick Start

### Option 1 - single command 

If you have Rust installed I think you can run

```
cargo install --git https://github.com/leomcl/swifi
```

### Option 2 - from source

1. Clone repo
```
git clone [https://github.com/leomclaughlin/swifi.git](https://github.com/leomclaughlin/swifi.git)
cd swifi
```
2. Build and install
```
cargo install --path .
```

## Ussage

```
swifi --help 
```
